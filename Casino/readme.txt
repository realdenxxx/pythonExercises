Условия задачи:

На колесе рулетки карманы пронумерованны от 0 до 36. Ниже приведены цвета карманов:

- карман 0 - зелёный
- карман с 1 по 10 с нечётным номером цвет - красный, с чётным - чёрный;
- карман с 11 по 18 с нечётным номером цвет - чёрный, с чётным - красный;
- карман с 19 по 28 с нечётным номером цвет - красный, с чётным - чёрный;
- карман с 28 по 36 с нечётным номером цвет - чёрный, с чётным - красный.

Программа просит пользователя ввести номер кармана и показывает цвет кармана.
если номер карман лежит вне диапазона от 0 до 36 программа выводит сообзение об ошибке.
